Custom Language Keywords

                        

                            
maano

                            
Declares a new variable.

                            
maano x = 10

                            
Example: Creates a variable x with value 10

                        

                        

                            
likho

                            
Prints output to console.

                            
likho "Hello World"

                            
Example: Prints "Hello World" to console

                        

                        

                            
agar

                            
Conditional statement (if).

                            
agar x > 5 {
    likho "x is greater than 5"
}

                            
Example: Prints message if condition is true

                        

                        

                            
warna

                            
Else statement

                            
agar x > 5 {
    likho "x is greater than 5"
} warna {
    likho "x is 5 or less"
}

                            
Example: Prints different messages based on condition

                        

                        

                            
jabtak

                            
While loop.

                            
jabtak x < 10 {
    likho x
    x = x + 1
}

                            
Example: Prints numbers from x to 9
